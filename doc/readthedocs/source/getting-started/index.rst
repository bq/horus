Getting started
===============

Horus is a multiplatform application to experiment with the `open 3D scanner Ciclop`_.

It provides a graphical interface that allows you to connect the scanner, control its devices, set scanning parameters, calibrate the scanner and scan 3D objects with Ciclop. It also includes a 3D scene of the point cloud obtained in real time.

.. image:: ../_static/getting-started/main-window.png

It was created by `bqlabs`_, the Department of Innovation and Robotics at `BQ`_, developed in `Python`_ and released under the `GPLv2`_ license.
